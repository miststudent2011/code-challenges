# [PRIME PALINDROME]

## CHALLENGE DESCRIPTION:

Write a program which determines the largest prime palindrome less than 1000.

## INPUT SAMPLE:

There is no input for this program.

## OUTPUT SAMPLE:

Print to stdout the largest prime palindrome less than 1000.

```
929
```
